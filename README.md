### 1. Function: ###
   Created to manage contact information
   
   Able to implement a persistent storage application: create, read, update and delete. (CRUD)

### 2. Languages: ###

   Front-end: *HTML*

   Back-end: *Python*

   Database: *MySQL*

   Library and framework used: *Bootstrap, JQuery, Flask*
